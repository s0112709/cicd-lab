# 2026 CI/CD Homework

## 作業說明

以此專案為基礎, fork 出自己的 cicd-lab

https://github.com/yubinTW/cicd-lab

## 基本要求

### 請完成以下項目：

在專案中新增 .github/workflows/ci\_{學號}.yaml 檔案

- pipeline 需要在 push 時自動執行 (20%)
- pipeline 中**至少**包含： (20%)
  - TypeScript typecheck
  - Prettier check
  - Test
- 任一檢查失敗時，GitHub Actions pipeline 應顯示失敗 (20%)
- 將測試結果顯示於 GitHub actions 結果頁面 (20%)
  - 可自己實作或使用現成的 GitHub actions
  - 參考: https://github.com/marketplace?type=actions
  - 可搭配修改專案中的 source code/ config
- 報告中詳加說明實作方式使用工具與策略 (20%)

## 繳交內容

### 完成基本要求的相簿，請繳交一份簡短報告，內容包含：

- CI Pipeline 說明
  - 貼上 .github/workflows/ci\_{學號}.yaml 的主要內容
  - 說明你設計的 pipeline
- CI 執行結果截圖
  - 至少一張成功執行的截圖
  - 附上 GitHub Actions 頁面的結果畫面
- 失敗案例說明
  - 請故意製造一個錯誤，例如：
    - TypeScript 型別錯誤
    - Prettier 格式錯誤
    - 測試失敗
  - 附上 pipeline failed 的截圖, 說明錯誤原因與修正方式
- 繳交格式: PDF 檔
- 檔案標題: 學號*姓名\_CICD*作業.pdf
